# 🏦 LendX Finance — Loan Portfolio Performance & Risk Analytics

> **A full-stack Business Intelligence project** built with **PostgreSQL + Power BI** to analyze loan performance, credit risk, and business profitability.

---

## 📌 Project Overview

LendX Finance is a fictional multi-segment lending company offering Personal, Home, Auto, Business, and BNPL loans across India.

### 🔹 What this project does:
- Designed a **production-ready PostgreSQL database**
- Built **30+ SQL queries** for KPIs & risk analysis
- Created an **interactive Power BI dashboard (7+ pages)**

---

## 📊 Key Metrics

| Metric | Value |
|-------|------|
| Total Revenue | ₹386.68M |
| Total Loan Book | ₹7.11B |
| Total Loans | 4,000 |
| Customers | 981 |
| Default Rate | 9.2% |

## 📉 Business Problem

- Rising default rates impacting profitability  
- High exposure to **low credit score customers**  
- No unified risk visibility across segments

## 🔍 SQL Analytics

- KPI Framework (Revenue, Default Rate, NPA, Yield)
- Portfolio Distribution Analysis
- Customer Risk Segmentation
- Regional & Branch Performance
- Advanced SQL (Window Functions)

---

## 🔑 Key Insights

- Portfolio is **loss-making due to high defaults**
- **BNPL loans = highest risk segment**
- **Home loans = safest & most profitable**
- Low credit score customers dominate portfolio

---

## 🚀 Business Recommendations

- Tighten approvals for **BNPL loans**
- Focus on **low-risk segments (Home Loans)**
- Apply **credit score threshold ≥ 650**
- Build **early warning system for defaults**
